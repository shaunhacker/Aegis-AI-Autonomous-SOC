# Aegis-AI-Autonomous-SOC
Project Aegis-AI is a high-tech SOC lab showcasing the future of autonomous defense. It integrates Microsoft Sentinel and NGFWs with Claude and Gemini to automate triage, log analysis, and firewall policy updates. By bridging SOAR workflows with AI, it transforms manual monitoring into a self-healing security ecosystem. Developed by Shaunhacker.
